# Code of Conduct

## Our Standards

This project is open to everyone. We expect participants to:

- Be respectful and constructive in discussions
- Accept feedback gracefully and offer it thoughtfully
- Focus on what is best for the project and its users
- Avoid personal attacks, harassment, and discriminatory language

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other venue where contributors represent the project.

## Enforcement

Instances of unacceptable behavior may be reported to Xavier Puspus at xpuspus@gmail.com. All reports will be reviewed and investigated promptly and confidentially.

Maintainers have the right to remove, edit, or reject contributions that violate these standards, and to ban contributors who engage in harmful behavior.
